How It Works

Every step runs autonomously. You give it a city name, it handles the rest.

1

Discover Clothing Stores

Queries OpenStreetMap, Yelp, or Google Places for clothing stores near a target city. Found 56 stores in Los Angeles in under 10 seconds.

Free — OpenStreetMap / Yelp
2

Qualify & Score Each Store

Scores each store 0–100 based on website presence, contact info, address completeness, and social media absence. Filters out chains and low-quality leads.

Free — Custom scoring engine
3

Scrape Clothing Images

Visits the store's actual website and extracts product images using BeautifulSoup. Targets product thumbnails, listing photos, and catalog images.

Free — BeautifulSoup scraper
4

AI Picks the Best Garment

Sends all scraped images to Groq Vision AI (Llama 4 Scout). Scores each garment on cinematic impact, color, silhouette, and uniqueness. Picks the winner.

Free — Groq Llama 4 Scout Vision
5

Generate Cinematic Video Reel

Uploads the winning garment to WaveSpeedAI's WAN 2.7 model. Generates a 5-second cinematic animation. With Segmind try-on enabled, the garment appears on a real model first.

~$0.06 — WaveSpeedAI WAN 2.7 Paid upgrade: model try-on via Replicate IDM-VTON
6

Design Print-Ready Postcard

Creates a 4×6" postcard at 300 DPI using Pillow. Front shows the garment with personalised headline. Back includes store name, a QR code linking to the video, and a call to action.

Free — Pillow + qrcode
7

Physically Mail It

Sends the postcard via Lob API. Lob prints, addresses, stamps, and mails it to the store owner's address. Returns a tracking ID. Test mode is free (300 cards).

Free trial — Lob API (300 test cards) Live: ~$0.80/postcard

Paid vs Free Path

We built and tested both. Here is the honest comparison.

Free Path

Bootstrap Mode

~$0.10 / store
  • OpenStreetMap store discovery (56 stores found, zero cost)
  • Groq Llama 4 Vision garment selection (free tier)
  • WaveSpeedAI WAN 2.7 video ($1 free signup credit)
  • Lob test mode mailing (300 free postcards)
  • Postcard design with Pillow (fully free)
  • Try-on quality limited (Segmind 406 in testing)
  • Less store data richness vs Google
  • Some APIs blocked by Indian ISPs (HuggingFace, fal.ai)
Paid Path

Production Grade

~$0.90 / store
  • Google Places API — phone, hours, owner name, photos
  • Replicate IDM-VTON — garment on real model photo
  • Replicate Stable Video Diffusion — cinematic model reel
  • Lob live mode — real tracked physical mail delivery
  • Claude claude-sonnet-5 vision — best garment reasoning

Errors Hit and Fixed

Everything that broke during development, and how we fixed it. Real debugging, not a happy path demo.

ErrorRoot CauseFix Applied
HuggingFace DNS fail Indian ISP blocks api-inference.huggingface.co at routing level — DNS change to 8.8.8.8 didn't help Switched to WaveSpeedAI (globally accessible)
fal.ai 403 / DNS blocked VPN IP blocked by fal.ai; storage.fal.run also blocked by ISP Switched to WaveSpeedAI; added tmpfiles.org upload step
Replicate 402 Free trial credits exhausted Added WaveSpeed + Segmind as free alternatives
Segmind IDM-VTON 406 Oversized base64 payload (~2MB) exceeding server limit Resize images to 768px max with Pillow before encoding
WaveSpeed 400 (480p) API only accepts "720p" or "1080p" — not "480p" Changed resolution to "720p"
WaveSpeed connection aborted Raw PNG base64 payload (~2MB) caused server to close connection Upload image to tmpfiles.org first, pass HTTPS URL instead
Lob 422 (size) Sent "6x4" — Lob only accepts "4x6" Fixed size string to "4x6"
Lob 422 (use_type) Required field missing from API payload Added use_type: "marketing" to request
OSM 0 qualified stores All OSM stores have rating=0; threshold was 50 — nothing passed Lower threshold to 15 when no stores have ratings
Groq 400 (deprecated model) llama-3.2-90b-vision-preview deprecated by Groq Switched to meta-llama/llama-4-scout-17b-16e-instruct
Overpass 504 Main OSM server overloaded Added 2 mirror fallback URLs with auto-retry
Mailing without video Pipeline mailed postcards with broken QR codes when video failed Gate Steps 6+7 on video_path being set — skip if no video
